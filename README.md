# ESUMOON — 現代化 Web 數位體驗與網頁設計

ESUMOON 是一個具有現代視覺美學、流暢動效與智慧互動介面的網頁應用。

## ✨ 特色亮點

- 🌙 **Glassmorphism 玻璃擬態 UI**：暗色星空質感，搭配半透明面板與動態漸變光暈。
- 🎨 **自訂主題強調色**：支援一鍵切換深色/淺色模式，並提供 Indigo、Cyan、Emerald、Rose、Amber 等調色盤即時預覽。
- 🌌 **HTML5 Canvas 粒子背景**：具備跟隨滑鼠互動的動態粒子連線效果。
- ⏱️ **即時儀表板與狀態指標**：實時時鐘顯示、模擬記憶體與網路延遲監控。
- 📱 **100% 響應式佈局 (Responsive Design)**：完美適配桌面、平板與手機等各種螢幕尺寸。
- ⚡ **純前端零外在依賴**：基於 HTML5、CSS3 與 ES6+ JavaScript，下載即可開啟運行。

## 📁 專案檔案結構

```
.
├── index.html   # 主網頁架構與內容
├── style.css    # HSL 色彩系統與 Glassmorphic 樣式
├── script.js    # 粒子動畫、主題切換、時鐘與作品篩選邏輯
└── README.md    # 專案說明文件
```

## 🚀 快速開始

1. 複製此專案庫：
   ```bash
   git clone https://github.com/hannwu20/Esumoon.git
   ```
2. 直接點擊開啟 `index.html` 即可在瀏覽器中體驗！
