# Esumoon 易善美 — 純鈦精緻壺器與飲食生活選品

Esumoon 易善美源自「用最簡單的方式，達到盡善盡美」的理念。專注於精緻純鈦養生壺器、德鐘鈦壺、八方鈦盒與飲食生活選品。以鈦為器，以茶修心，在一壼一食之間，品見生活真美。

## 🌟 網站重建與亮點

- 🫖 **德鐘鈦壺 x 八方鈦盒新品專區**：完整展示純鈦新品特色與千度結晶視覺美學。
- 🛡️ **純鈦 5 大核心優勢**：安全無毒抗酸鹼、無鐵鏽異味無重金屬溶出、不氧化高硬度、無需繁複保養、千度結晶古韻美學。
- 📜 **權威檢測報告與認證**：包含 SGS 99.9% 純鈦純度報告與重金屬零析出品質承諾。
- 🎨 **尊榮質感 UI 設計**：玻璃擬態 (Glassmorphism)、深/淺主題切換與沉浸式粒子氛圍動畫。
- ❓ **常見問題 (FAQ) 折疊選單**：收錄茶具保養、加熱兼容性與結晶紋理維護答疑。
- 📱 **響應式全裝置適配**：適配電腦、平板與手機等各式螢幕。
